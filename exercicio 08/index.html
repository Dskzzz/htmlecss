Principais formataçoes

Negrito / Destaque

Para colocar em negrito, usamos a tag B, porem ela nao é semantica.Ou seja, nao tem significado

Agora, uma tag que possue destaque sendo semantica, é a tag Strong

Italico / Destaque

Para colocar um termo em italico,usamos a tag i, porem ela nao é semantica. ou seja, nao tem significado

Para colocarmos um temos em italico sendo semantico, utilizamos a tag em

Texto marcado

Podemos tambem criar um texto marcado usando a tag Mark

Texto grande e pequeno

Para deixarmos o texto pequeno, usamos a tag small

Texto deletado

Para marcarmos um texto como lido mas nao considerado, usamos a tag del

Texto inserido

Para indicar que um texto foi inserido, usamos a tag ins

Texto sobrescrito

Para deixarmos as coisas sobrescritas como por exemplo x20+3=0, usamos a tag sup

Texto subscrito

Para deixarmos as coisas subscritas como por exemplo H2O, usamos a tag sub
